DO NOT RUN THESE COMMANDS IN YOUR INSTANCE!!!!!!!
DO NOT SIGN IN TO YOUR INSTANCE

0. Unzip the here.zip folder to your desktop. 


Then in git bash, run these command



1. sudo apt-get install software-properties-common
2. sudo apt-add-repository universe
3. sudo apt-get update
4. sudo apt-get install python3-dev python3-pip
5. sudo apt install python3
6. pip3 install --upgrade pip setuptools
7. pip install Flask --user
8. go to 'here' folder and run    FLASK_APP=main.py flask run
9. go to http://127.0.0.1:5000/
